# Train Booking

這是一個用於訂購火車票的程式，具有使用者介面。

## 安裝與設定

1. 安裝Python。
2. 下載這個專案的代碼。
3. 在專案的根目錄中執行 `python main.py`。

## 使用方式

啟動應用後，你可以在圖形界面中輸入你的訂票需求，包括日期、起點和終點等信息，然後點擊 "確定" 按鈕進行訂票。
